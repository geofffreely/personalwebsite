Local copies of your papers for the Research page.

How it works
------------
Each linked title on writings.html points to a PDF in this folder. The
filename is the entry id without a prefix ("pub-", "rev-", "book-", or "draft-"), e.g.:

  id="pub-LEECPA-5"     →  LEECPA-5.pdf
  id="rev-LEETCB-3"     →  LEETCB-3.pdf
  id="book-LEETIO-22"   →  LEETIO-22.pdf
  id="draft-LEESD"      →  LEESD.pdf

Add or replace PDFs here using those exact names. Until a file is present,
that link will return “not found” when clicked—add the PDF when you have it.

Upload this entire "papers" folder with the rest of the site when you deploy.
